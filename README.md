# WebWizards Task

Projekt wykonany w ramach praktyk w firmie WebWizards.

## Technologie

- Vue 3
- Nuxt 4
- SCSS

## Funkcjonalności

- pobieranie danych użytkownika z API,
- wyświetlanie danych użytkownika,
- ukrywanie i pokazywanie numeru telefonu,
- rozwijanie i zwijanie opisu,
- pobieranie danych użytkownika do pliku JSON,
- responsywny interfejs użytkownika.

## Uruchomienie projektu

```bash
npm install
npm run dev
```
